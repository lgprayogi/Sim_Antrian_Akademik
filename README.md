# Sim_Antrian_Akademik
## Petunjuk Penggunaan

Sistem simulasi ini memiliki menu sebagai berikut,
1. Tambah mahasiswa (manual)                 
   Menu ini akan menambahkan data antrean berdasarkan data yang di input oleh pengguna.
2. Tampilkan antrean          
   Menu ini akan menampilkan antrean yang ada di dalam sistem berdasarkan jenis algoritma (fcfs atau prioritas)
3. Proses satu mahasiswa              
   Menu ini akan memilih satu data antrean yang sesuai dengan jenis algoritma untuk diproses.
4. Proses semua antrean           
   Menu ini akan memroses semua antran yang ada di dalam sistem dengan dua algoritma.
5. Generate antrean otomatis          
   Menu ini akan membuat data antrean otomatis sesuai jumlah data yang di input pengguna.
6. Hapus antrean          
   Menu ini akan menghapus semua antrean yang ada di dalam sistem
7. Keluar          
   Menu ini akan mengeluarkan pengguna dari simulasi.
